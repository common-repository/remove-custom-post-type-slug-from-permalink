== Description ==

This plugin is use to remove your slug in custom post type and default post type.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `remove-custom-post-type-slug-from-permalink.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on `Remove custom post type slug from URL` and add your settings
